# Resume Builder Template

A ready-to-use React + Vite + TailwindCSS (v4) template for building your own resume.

## 🚀 How to Start

```bash
git clone <your-repo-link>
cd resume-builder-template
npm install
npm run dev
```

## 🛠 What's Included
- React + TypeScript
- Tailwind CSS v4
- Preconfigured Vite
- Auto setup script (`setup.bat`)
